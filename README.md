# 💰 LoanIQ — Smart Loan Calculator

LoanIQ is a modern and easy-to-use loan calculator web application that helps users calculate EMIs, compare loan options, manage multiple loans, and estimate tax benefits — all in one place.

Built with clean UI and practical finance tools, this project focuses on real-world usability and smooth user experience.

## ▶️ Live Demo

🔗 https://monu655.github.io/Loan_IQ/

## ✨ Key Features

### 🧮 Loan Calculator

* Instant EMI calculation
* Total payment and interest breakdown
* Interest percentage insights
* Loan-free year prediction
* Principal vs Interest visualization

### 📊 Detailed Breakdown

* Year-wise amortization schedule
* Monthly payment breakdown
* Remaining balance tracking
* Loan completion progress

### 📁 Loan Manager

* Save multiple loans
* Track total borrowed amount
* Combined EMI & interest overview
* Easy loan removal

### ⚖️ Loan Comparison

* Compare EMI for different tenures
* Identify lowest EMI option
* Smart repayment suggestions

### 🧾 Tax Benefit Estimator

* Home loan tax savings calculation
* Section 24B & 80C benefits
* Old vs New tax regime comparison
* Effective EMI after tax savings

### 💸 Affordability Checker

* Maximum safe EMI calculation
* Eligible loan amount estimate
* Financial health indicator
* Disposable income analysis

### 📱 Responsive Design

* Mobile-friendly interface
* Clean and modern layout
* Smooth tab navigation
  
## 🛠️ Tech Stack

* **HTML5** — Structure
* **CSS3** — Styling & Responsive Design
* **Vanilla JavaScript** — Business Logic & Calculations

## 🎯 Project Objective

This project was developed to strengthen practical JavaScript skills and build a real-world financial utility tool.
It also serves as a portfolio project for frontend and web developer roles.
